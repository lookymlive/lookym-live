# LOOKYM - Project Description

## Overview
LOOKYM is an innovative Marketplace for commercial storefront videos, a multichannel platform designed to uniquely connect businesses and users. Businesses can showcase their products through their storefronts, while users have the opportunity to search, discover, and interact with their favorite storefronts. Additionally, they can upload their 3D figures and try on products. Direct communication with brands and the ability to make purchases directly from the platform.

## Technology Stack
- Frontend: React Native with Expo
- Backend: Supabase for authentication, database, and real-time features
- Media Storage: Cloudinary for video hosting and streaming
- State Management: Zustand with persistence
- Styling: React Native StyleSheet

## Key Features
1. Authentication with user/business roles
2. Short-form video upload and viewing
3. Direct messaging between users and businesses
4. Video interactions (likes, comments, saves)
5. User and business profiles
6. Search and discovery

## Project Structure
- /app: Screens and navigation (Expo Router)
- /components: Reusable UI components
- /store: Zustand stores for state management
- /utils: Utility functions and API clients
- /types: TypeScript type definitions
- /mocks: Mock data for development
- /docs: Project documentation
